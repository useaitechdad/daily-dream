## What this release is

The complete specification handed to an AI coding agent (Antigravity) before any code was written. This release is deliberately empty of implementation — it shows what "done" looks like for a spec-only phase.

## What's in this folder

- `PRD.md` — product requirements for the Mii designer
- `project-rules.md` — always-on coding rules for the agent
- `mii-schema.md` — canonical Mii data shape
- `KICKOFF.md` — the exact prompt pasted into the agent
- `WHAT-CHANGED.md` — notes on what this release establishes

## The story

My daughter and I wanted to build something in the spirit of Tomodachi Life. Instead of jumping straight into code, we spent a weekend writing a tight spec. The hypothesis: giving an AI agent a good spec matters more than writing clever prompts. The rest of the series tests that hypothesis.

## How to use these files yourself

1. Adapt the scope in `PRD.md` to your project
2. Open your AI coding agent (Antigravity, Claude Code, Cursor, Windsurf, etc.)
3. Paste `KICKOFF.md`
4. Review the plan artifact before approving
